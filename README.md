## 注册准备

官网： https://chat.openai.com/

准备：可以访问`OpenAI` 官网，但ChatGPT未对中国大陆开放，且需要境外手机号才能注册，因此需要魔法伪装为境外用户，费用一般<=1刀乐。



备注： 注册并成功进入 ChatGPT 后，无需联网即可正常对话。 在 Cookie 有效情况下，之后可能直连进入。
